## Le contexte

« Wonmafé » signifie approximativement « chez nous » en langue locale — un nom choisi pour ancrer le projet dans le quotidien plutôt que dans le jargon technique. L'idée : rendre lisibles, pour n'importe quel citoyen, les écarts de développement entre régions guinéennes.

## L'approche

Les données publiques disponibles (infrastructures, accès aux services de base, indicateurs socio-économiques) sont consolidées côté Django/PostgreSQL puis exposées sous forme de visualisations Chart.js pensées pour rester lisibles sur mobile — le principal terminal d'accès à internet en Guinée. Chaque graphique est accompagné d'un texte d'explication en français simple, sans jargon statistique.

## Ce qui compte

Wonmafé reste un projet personnel, hors mandat d'une organisation commanditaire, mené pour explorer ce que la donnée publique peut raconter quand elle est mise en forme pour le grand public plutôt que pour des experts.
